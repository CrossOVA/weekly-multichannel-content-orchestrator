# Еженедельный мультиканальный контент-оркестратор

Сценарий n8n для автоматической генерации и публикации контента по недельной редакционной сетке.

## Что делает сценарий

Сценарий путём использования динамического промпта, реализованного на code-узле, сам определяет день недели и выбирает подходящий формат публикации:

- опрос;
- мем;
- мотивационный пост;
- образовательный пост;
- дискуссионный пост.

После этого он генерирует текст для нескольких каналов, проверяет повторы тем через SQL-БД, готовит идею для изображения и само изображение,, публикует контент и сохраняет результат.

## Что реализовано

- расписание публикаций по дням недели;
- разные промпты под разные типы постов;
- генерация отличающихся текстов для X, Telegram и Reddit;
- генерация идеи для изображения и самого изображения;
- проверка уже использованных тем;
- повторная генерация, если тема уже была;
- запись опубликованных тем в базу;
- публикация через внешние API;
- сохранение результата в таблицу;
- Telegram-уведомления администратора об ошибках и статусах.
